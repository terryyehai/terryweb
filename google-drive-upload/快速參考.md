# ⚡ 快速參考卡

## 🎯 三步驟完成設定

### 1️⃣ 上傳檔案
```
google-drive-upload/gallery-data.json
↓ 拖曳上傳到 ↓
https://drive.google.com/drive/u/2/folders/1qm0SO7-XAOlYITnvrvZGQitwnOKZN83j
```

### 2️⃣ 設為公開
```
右鍵 → 取得連結 → 知道連結的任何人都可以檢視 ✅
```

### 3️⃣ 網站設定
```
http://localhost:4175/#/settings
→ 貼上連結 → 測試連線 → 儲存設定
```

## 🆘 故障排除

**測試連線失敗？**
→ 檢查檔案是否設為公開分享

**相簿空白？**
→ 清除快取 + 重新整理（Ctrl+F5）

**圖片無法顯示？**
→ 檢查 F12 Console 錯誤訊息
