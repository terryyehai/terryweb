## 🎯 快速開始

### 只需 3 步驟：

1️⃣ **上傳 `gallery-data.json`**
   ```
   拖曳到您的 Google Drive 資料夾
   ```

2️⃣ **設為公開分享**
   ```
   右鍵 → 取得連結 → 知道連結的任何人都可以檢視
   ```

3️⃣ **在網站設定**
   ```
   http://localhost:4175/#/settings
   貼上連結 → 測試 → 儲存
   ```

## ⚠️ 重要提醒

### 關於圖片來源

> 目前 JSON 中的圖片連結使用 **Unsplash** 和 **Mixkit** 的免費資源
> 
> 這是為了讓您可以**立即測試功能**

### 如何替換成自己的照片？

稍後您可以：

1. **上傳照片**到 Google Drive
2. **取得分享連結**（設為公開）
3. **編輯 JSON 檔案**，替換 `thumbnail`、`fullsize`、`videoUrl` 的連結
4. **重新上傳** JSON 檔案
5. **清除網站快取**並重新整理
