## 🎯 上傳步驟（3 步驟完成）

### ✅ 步驟 1：上傳 JSON 檔案

1. **開啟您的 Google Drive 資料夾**：
   - 點擊上方連結，或直接前往您的 Google Drive
   - 確認您已登入正確的帳號（u/2）

2. **上傳 JSON 檔案**：
   - 在專案資料夾找到：`google-drive-upload/gallery-data.json`
   - 拖曳檔案到 Google Drive 資料夾，或
   - 點擊「新增」→「檔案上傳」→ 選擇 `gallery-data.json`

3. **確認上傳成功**：
   - 在 Google Drive 中看到 `gallery-data.json` 檔案

---

### ✅ 步驟 2：設定為公開分享

1. **選取檔案**：
   - 在 Google Drive 中點選 `gallery-data.json`

2. **取得連結**：
   - 右鍵點擊 → **「取得連結」** 或點擊分享圖示 🔗

3. **變更權限**：
   - 點擊「限制」下拉選單
   - 選擇：**「知道連結的任何人都可以檢視」** ✅
   - 點擊「完成」

4. **複製連結**：
   - 點擊「複製連結」按鈕
   - 儲存此連結，稍後會用到

---

### ✅ 步驟 3：在網站設定

1. **開啟網站設定頁面**：
   ```
   http://localhost:4175/#/settings
   ```

2. **貼上連結**：
   - 在「Google Drive JSON 檔案連結」欄位貼上剛才複製的連結

3. **測試連線**：
   - 點擊「🔍 測試連線」按鈕
   - 應該會看到：**「✅ 連線成功！找到 12 個項目」**

4. **儲存設定**：
   - 點擊「💾 儲存設定」按鈕

5. **查看結果**：
   - 重新整理頁面（F5）
   - 前往「相簿」頁面
   - 應該會看到 12 個台灣景點的照片和影片

## ⚠️ 重要提醒

### 關於圖片來源
> **目前使用的是 Unsplash 和 Mixkit 的外部圖片/影片連結**
> 
> 這是為了讓您可以立即看到效果。稍後您可以：
> 1. 上傳自己的照片到 Google Drive
> 2. 取得每張照片的分享連結
> 3. 編輯 JSON 檔案，替換成您自己的圖片連結

### 如何使用自己的照片

當您準備好替換成自己的照片時：

1. **上傳照片到 Google Drive**：
   - 將照片上傳到同一個資料夾
   - 建議每張照片準備兩個版本：
     - 縮圖版：600x400 px（≤ 500 KB）
     - 完整版：1920x1080 px（≤ 3 MB）

2. **設定為公開分享**：
   - 選取照片 → 右鍵 → 取得連結
   - 改為「知道連結的任何人都可以檢視」

3. **編輯 JSON 檔案**：
   - 下載 `gallery-data.json` 並編輯
   - 將 `thumbnail` 和 `fullsize` 的連結替換成您的照片連結
   - 格式：`https://drive.google.com/file/d/YOUR_FILE_ID/view`

4. **重新上傳 JSON 檔案**：
   - 覆蓋原本的 `gallery-data.json`
   - 在網站清除快取
   - 重新整理頁面

## 🔧 故障排除

### ❌ 測試連線失敗

**檢查清單**：
- [ ] JSON 檔案是否已上傳到 Google Drive？
- [ ] 檔案是否設定為「知道連結的任何人都可以檢視」？
- [ ] 複製的連結是否完整？
- [ ] 網路連線是否正常？

**解決方式**：
1. 重新確認檔案權限
2. 嘗試在瀏覽器無痕模式開啟連結，看是否能下載檔案
3. 檢查瀏覽器 Console（F12）的錯誤訊息

### ❌ 相簿頁面空白

**可能原因**：
- 快取未清除
- JSON 連結未儲存

**解決方式**：
1. 前往「設定」頁面
2. 點擊「清除快取」
3. 確認「目前資料來源」顯示您的連結
4. 重新整理頁面（F5 或 Ctrl+F5）
